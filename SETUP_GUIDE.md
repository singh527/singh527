## ðŸ”§ Troubleshooting

### Stats Not Showing?
- Make sure your repositories are public
- Wait a few minutes for GitHub to update
- Check if the stats services are online

### Snake Animation Not Working?
- Ensure GitHub Actions is enabled
- Check workflow permissions in Settings
- Run the workflow manually first

### Banner Not Displaying?
- Make sure `banner.png` is in the root of your repository
- Check the file path in README.md matches exactly
